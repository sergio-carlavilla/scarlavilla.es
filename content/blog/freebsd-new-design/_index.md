After several years of development, we’ve launched the new design for the FreeBSD website.
With this redesign we wanted to give the main FreeBSD website a facelift.

{{< figure src="freebsd-website.png" width="100%" >}}

What improvements have we implemented?

* Showcase FreeBSD’s main features
* Light/dark mode
* Responsive design (Yes, responsive design in 2026 :D)

Is this the end of the road? No, the idea is to keep improving the website.

The next steps will be:

* Update the about and features pages
* Improve the downloads page
* Try to translate FreeBSD into as many languages as possible
